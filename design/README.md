# UI/UX & Visual Prototypes

This folder contains the structural design assets and UI layout files for the BioFuse project dashboard. The primary user experiences and interface elements were constructed using Figma to visually map out how a user interacts with the digital twin system.

### Asset Overview: Figma Source Packages
- **Layout & Prototyping:** Contains high-fidelity UI design files detailing the web app interface, simulator control boards, and responsive layouts.
- **State Change Renders:** Wireframes and visual components demonstrating the interface shifts as the indicator transitions from 'Optimal' (Safe) to 'Critical' (Warning states).
- **Component Design:** Visual specifications for the dashboard cards, temperature adjustment sliders, and data visibility modules.

*Note: These digital assets serve as the creative design blueprint for the interactive front-end, aligning software control interfaces with the structural logic of the safety tracking system.*
