# BioFuse

### Project Overview
BioFuse is a non-invasive, external thermal indicator for medicine vials. 

### How it Works
It uses two reactive solutions (Solution A and Solution B) separated by a thin wax barrier. 
- When the temperature exceeds a safe limit, the wax melts.
- The solutions mix and change color to warn the user.
- **Safety Note:** This indicator is external and never touches the medicine.
